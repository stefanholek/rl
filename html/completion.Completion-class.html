Class Completion

object --+

        Completion

Interface to the active readline completion.

This class is not intended for instantiation beyond the one completion object in this module. Applications wanting to use the Completion interface will typically import the completion object and use its properties and methods to implement custom completions.

Example:

from completion import completion

def complete(text):
    completion.append_character = '@'
    return completion.complete_username(text)
