Class Completer

source code

object --+
         |
        Completer

Interface to the readline completer configuration.

This class is not intended for instantiation beyond the one completer object in this module. Applications wanting to use the Completer interface will typically import the completer object and use its properties and methods to configure readline.

Example:

from completion import completer

completer.quote_characters = '"\''
completer.query_items = 100
completer.parse_and_bind('tab: complete')
